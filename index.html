About

Carl, aka Cid, is a professional software engineer from Seattle, WA. Although trained as a fullstack developer, he specializes in frontend solutions. His passion for the craft runs wide, leading to deep curiosities regarding all things tech, including productivity, work management, project management, programming paradigms, problem-solving patterns, design patterns, web accessibility, and many, many more.

This website is a learning resource for its maintainer and its visitors. For the maintainer, it is intentionally over-engineered. It only needs HTML, CSS, and JavaScript to provide value to visitors in the form of educational content about various software engineering topics. However, the actual implementation of this website goes further, all in the name of hands-on experience with technologies in use out in the wild.

To a degree, frameworks, libraries, abstractions, and tools will be avoided in this website's implementation, emphasizing software engineering fundamentals and pragmatism. So, over time, this website's visual appearance will change to become more appealing, but will probably never do anything "flashy" without good reason.

Coming soon is a blog that will document the development of this website and provide readers with content about all sorts of software engineering topics. First up will be a round of posts about data structures and algorithms.
